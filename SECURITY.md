# Security Policy

Reporting a Vulnerability
- Please email security@nabi.example (replace with your address) with details and a proof of concept if possible.
- Use the subject: [SECURITY] Short description.
- We will acknowledge within 72 hours, triage, and coordinate a fix and disclosure timeline with you.

Handling Process
1. Triage and reproduce.
2. Assign CVSS score and severity.
3. Prepare patches for supported branches.
4. Coordinate release with release notes and credit.

Supported Versions
- See docs/RELEASES.md for LTS and support windows.

Key Management
- If you prefer encryption, provide your PGP key or request ours in the initial email.

Disclosure
- We practice responsible disclosure. Please do not create public issues for vulnerabilities.
